Malone has a bug filing page on which you can specify the distro in
addition to the source package in which the bug occurs.

You must be logged-in to file a bug. After you filed the bug you get
redirected to the bug page.

    >>> browser.addHeader("Authorization", "Basic foo.bar@canonical.com:test")
    >>> browser.open("http://localhost:9000/bugs/bugs/+package")
    >>> browser.getControl(name="field.distribution").value = ["ubuntu"]
    >>> browser.getControl(name='packagename_option').value = ['choose']
    >>> browser.getControl(name="field.packagename").value = "mozilla-firefox"
    >>> browser.getControl(name="field.title").value = "A brief bug title"
    >>> browser.getControl(name="field.comment").value = (
    ...     "A really brief bug description.")
    >>> browser.getControl("Submit Bug Report").click()
    >>> print browser.url
    http://localhost:9000/ubuntu/+source/mozilla-firefox/+bug/11

Filing of a bug against a distribution that has no current releases is
handled, but specially -- the package name is ignored and the bug's
description has a special comment appended to it.

    >>> browser.open("http://localhost:9000/bugs/bugs/+package")
    >>> browser.getControl(name="field.distribution").value = ["gentoo"]
    >>> browser.getControl(name='packagename_option').value = ['choose']
    >>> browser.getControl(name="field.packagename").value = "mozilla-firefox"
    >>> browser.getControl(name="field.title").value = "A brief bug title"
    >>> browser.getControl(name="field.comment").value = (
    ...     "A really brief bug description.")
    >>> browser.getControl("Submit Bug Report").click()
    >>> print browser.url
    http://localhost:9000/gentoo/+bug/12
