= Launchpad Bug Statistics =

On the Bugs front page there is some statistics shown, showing:

  * number of bugs that are filed in Launchpad
  * number of links to external bugs
  * number of external bug trackers registered
  * number of CVE bug links

  >>> anon_browser.open('http://bugs.launchpad.dev/')
  >>> stat_portlet = find_portlet(anon_browser.contents, 'Malone statistics')
  >>> for text in stat_portlet.div.findAll(text=True):
  ...   print text
  10 Bugs
  10 Remote Bugs
  5 Bug Trackers
  0 Web links
  22 Bug Tasks
  2 CVE Bug Links
