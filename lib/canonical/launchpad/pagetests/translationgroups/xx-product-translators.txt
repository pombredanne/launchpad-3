The translation work for a product can be assigned to a translation group.

    >>> admin_browser.open(
    ... 'http://translations.launchpad.dev/evolution/+changetranslators')
    >>> admin_browser.title
    'Set translation group for The Evolution Groupware Application'
    >>> content = find_main_content(admin_browser.contents)
    >>> content.findAll('h1')
    [<h1>Select a new translation group</h1>]
    >>> hint = find_tag_by_id(admin_browser.contents, 'form_extra_info')
    >>> print hint
    <p...
    Select the translation group that will be managing...
    ...The Evolution Groupware Application...
    ...

To create a new translation group, the user has to go to the answers page
for Launchpad translations and submit a question.

    >>> answers_link = admin_browser.getLink(
    ... url='http://answers.launchpad.dev/rosetta/+addquestion')
    >>> answers_link is not None
    True
    >>> answers_link.click()
    >>> admin_browser.title
    'Ask a question about Launchpad Translations'
