= Top contributors =

For products, projects and distributions, we have a portlet which displays
the top 5 overall contributors of that KarmaContext
(product/project/distribution).  Also, the menu contains a link to the
+topcontributors page, where we list the top overall contributors and the
top contributors by category (Bugs, Specs, Translation, etc).

== Top contributors of a distribution ==

    # The top-contributors portlet has been removed from the distros page
    # because it was causing that page to timeout. That's why, for now, we're
    # accessing it directly here.
    >>> anon_browser.open(
    ...     'http://launchpad.dev/ubuntu/+portlet-top-contributors')
    >>> anon_browser.url
    'http://launchpad.dev/ubuntu/+portlet-top-contributors'

    >>> print anon_browser.contents
    <div class="portlet" id="...">...
    ...Top contributors...
    ...Mark Shuttleworth...
    ...Foo Bar...
    ...Sample Person...

    >>> anon_browser.open('http://launchpad.dev/ubuntu/')
    >>> anon_browser.getLink('List top contributors').click()
    >>> anon_browser.url
    'http://launchpad.dev/ubuntu/+topcontributors'
    >>> print anon_browser.contents
    <!DOCTYPE...
    ...Overall...
    ...Mark Shuttleworth...
    ...Foo Bar...
    ...Sample Person...
    ...By category...
    ...Bug Management...
    ...Foo Bar...
    ...Sample Person...
    ...Specification Tracking...
    ...Mark Shuttleworth...


== Top contributors of a product ==

    >>> anon_browser.open('http://launchpad.dev/firefox')
    >>> print find_portlet(
    ...     anon_browser.contents, 'Top contributors').renderContents()
    <BLANKLINE>
    <h2>Top contributors</h2>
    ...Sample Person...
    ...Mark Shuttleworth...
    ...Foo Bar...

    >>> anon_browser.getLink('List top contributors').click()
    >>> anon_browser.url
    'http://launchpad.dev/firefox/+topcontributors'

    >>> contents = anon_browser.contents
    >>> print find_tag_by_id(contents, 'overall_top').renderContents()
    <BLANKLINE>
    ...Sample Person...
    ...Mark Shuttleworth...
    ...Foo Bar...

    >>> print find_tag_by_id(contents, 'Bug Management').renderContents()
    <BLANKLINE>
    ...Sample Person...
    ...Foo Bar...

    >>> print find_tag_by_id(
    ...     contents, 'Specification Tracking').renderContents()
    <BLANKLINE>
    ...Mark Shuttleworth...


== Top contributors of a project ==

Anybody who contributes to a product of a given project is considered a
contributor of that project.

    >>> anon_browser.open('http://launchpad.dev/mozilla')
    >>> print find_portlet(
    ...     anon_browser.contents, 'Top contributors').renderContents()
    <BLANKLINE>
    <h2>Top contributors</h2>
    ...Sample Person...
    ...Mark Shuttleworth...
    ...Foo Bar...

    >>> anon_browser.url
