DistroSeries information edit.
This tests makes a modification in warty distroseries and check 
if the page is not broken.

Navigate to the edit page as admin

  >>> admin_browser.open('http://localhost/ubuntu/warty')
  >>> admin_browser.getLink('Change details').click()
  >>> admin_browser.url
  'http://localhost/ubuntu/warty/+edit'

Fill out the form and submit.

  >>> admin_browser.getControl("Display name").value = 'WartyII'
  >>> admin_browser.getControl("Title").value = 'The Warty Warthog II Release'
  >>> admin_browser.getControl("Summary").value = 'summary'
  >>> admin_browser.getControl("Description").value = 'description'
  >>> admin_browser.getControl("Change").click()
  >>> admin_browser.url
  'http://localhost/ubuntu/warty/+edit'
  >>> admin_browser.contents
  '...Updated on...'


Check +admin behaviour:

  >>> admin_browser.getLink("Administer").click()
  >>> admin_browser.url
  'http://localhost/ubuntu/warty/+admin'

Change changeslist field (supposed to be an email address)

  >>> admin_browser.getControl("Changeslist").value = 'foo@bar.com'
  >>> admin_browser.getControl("Change").click()
  >>> admin_browser.url
  'http://localhost/ubuntu/warty/+admin'
  >>> admin_browser.contents
  '...Updated on...'

Check if valid_email constraint is working:

  >>> admin_browser.getControl("Changeslist").value = 'foo bar'
  >>> admin_browser.getControl("Change").click()
  >>> admin_browser.url
  'http://localhost/ubuntu/warty/+admin'
  >>> admin_browser.contents
  '...Constraint not satisfied...'
