= Support Tracker Introduction Page =

The support tracker main page present a short introduction to the
application.

    >>> anon_browser.open('http://answers.launchpad.dev/')
    >>> print anon_browser.title
    Launchpad support tracker

It also displays an indication of recent activity by
displaying the number of requests tracked by the system.

    >>> print find_tag_by_id(
    ...     anon_browser.contents, 'requests-count').renderContents()
    The Answer Tracker contains 13 questions.

It also shows the 10 latest requests made:

    >>> latest_made_requests = find_tag_by_id(
    ...     anon_browser.contents, 'latest-requests-made')
    >>> print latest_made_requests.first('h2').renderContents()
    Latest questions asked
    >>> for row in latest_made_requests.fetch('tr', 'noted'):
    ...     row.first('a').renderContents()
    'Problemas de Impress\xc3\xa3o no Firefox'
    'Problema al recompilar kernel con soporte smp (doble-n\xc3\xbacleo)'
    'Continue playing after shutdown'
    'Installation failed'
    'Firefox loses focus and gets stuck'
    'Problem showing the SVG demo on W3C site'
    'Firefox cannot render Bank Site'

As well as the 10 latest requests solved:

    >>> latest_solved_requests = find_tag_by_id(
    ...     anon_browser.contents, 'latest-requests-resolved')
    >>> print latest_solved_requests.first('h2').renderContents()
    Latest questions solved
    >>> for row in latest_solved_requests.fetch('tr', 'noted'):
    ...     row.first('a').renderContents()
    'mailto: problem in webpage'
